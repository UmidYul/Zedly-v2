# Career Guidance — Профориентирование

**Файл:** `docs/03_features/career_guidance.md`  
**Приоритет:** Phase 1+  
**Статус:** Production Blueprint

---

## 1. Цель

Модуль определяет профиль склонностей ученика и даёт рекомендации:
- по предметам,
- по треку (STEM/гуманитарный/соц-экон и т.д.),
- по следующему шагу развития.

---

## 2. Архитектурное решение

- Это отдельный модуль поверх Test Engine.
- Тестовая часть переиспользует `tests/questions/test_sessions`.
- Профили и рекомендации хранятся в отдельных таблицах `career_profiles`, `career_recommendations`, `career_reports`.

---

## 3. Доступ

Видят:
- Администрация школы (Director),
- Psychologist,
- Классный руководитель,
- Parent (своего ребёнка).

Ограничение:
- доступ только в пределах `school_id`.

---

## 4. Поток

1. Ученик проходит `career_guidance` тест.
2. Rules engine рассчитывает базовый профиль.
3. AI enrichment формирует персональные рекомендации.
4. Генерируется PDF-отчёт.
5. Отчёт сохраняется и доступен разрешённым ролям.

---

## 5. Интеграция AI

- AI не заменяет rules engine, а обогащает рекомендации.
- Нужен guardrail:
  - фиксированный prompt-template,
  - allowlist формулировок,
  - запрет медицинских диагнозов.
- Все AI outputs логируются с versioned prompt id.
